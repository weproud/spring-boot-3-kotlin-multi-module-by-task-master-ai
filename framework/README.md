# Framework 모듈

이 모듈은 외부 시스템과의 연동 및 인프라 관련 기능을 제공합니다.

## 하위 모듈

### client

외부 API 클라이언트를 포함합니다.

#### base

WebClient 공통 설정 및 기본 클라이언트 클래스를 제공합니다.

#### ai

AI API 연동을 위한 클라이언트를 제공합니다.

#### kakao

카카오 API 연동을 위한 클라이언트를 제공합니다.

### provider

#### jwt

JWT 토큰 생성 및 검증 기능을 제공합니다.

### redis

Redis 캐싱 및 세션 관리 기능을 제공합니다.
